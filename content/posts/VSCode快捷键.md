[VS Code 快捷键参考](https://www.cnblogs.com/jpfss/p/10956650.html")



```
ctrl + ~                # 终端显示和隐藏
shift + alt + up/down   # 复制当前行
Alt+ ↑ / ↓              # Move line up/down
shift + ctrl + k        # 删除当前行
```

```
ctrl + shift + f        # 全局查找
```
